Print & Input

  • print() displays output to the console.
  • input() gets user input as a string.
  • Use \n for new lines in strings.
  • String concatenation with + joins strings together.

Variables & Naming

  • Variables store data for later use.
  • Variable names should be descriptive and follow Python naming rules (letters, numbers, underscores; cannot start with a number).

String Operations

  • Strings are sequences of characters, defined with quotes.
  • Concatenation: "Hello" + "World" â†’ HelloWorld
  • Indexing: "Hello"[1] gives e.
  • len() returns the length of a string.

Data Types

  • str: String (text)
  • int: Integer (whole number)
  • float: Floating point (decimal number)
  • bool: Boolean (True or False)
  • Use underscores for readability in large numbers: 123_456_789

Type Checking & Casting

  • type() checks the data type of a value.
  • Type casting converts between types, e.g., int("123"), str(123).

Operators

  • Arithmetic: +, -, *, /, % (modulo), ** (power)
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not

Conditional Statements

  • if, elif, else control the flow based on conditions.
  • Indentation is crucial for defining code blocks.
  • Use comparison and logical operators in conditions.
  • Nested conditionals allow for more complex logic.

Loops

  • for loops iterate over sequences (like lists).
  • while loops repeat as long as a condition is true.
  • Indentation is required for the loop body.

Functions

  • Functions are defined with def and called by name.
  • They help organize code and avoid repetition.
  • Parameters allow functions to accept input values.
  • Indentation is required for the function body.

Lists & Indexing

  • Lists store multiple items in a single variable, defined with square brackets [].
  • Items can be accessed by index (starting at 0).
  • Lists can contain mixed data types.
  • Be careful of index out-of-range errors.

Random Module

  • Use import random to access randomization functions.
  • random.randint(a, b) returns a random integer between a and b (inclusive).
  • random.random() returns a random float between 0.0 (inclusive) and 1.0 (exclusive).
  • random.choice(list) selects a random item from a list.

Project Example: Hangman

  • Combines input, loops, conditionals, lists, and randomization.
  • Tracks user guesses and remaining lives.
  • Uses a loop to continue the game until win/lose condition is met.
  • Demonstrates how to build a simple game by combining multiple concepts.
